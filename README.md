AI Interview Coach

Overview

AI Interview Coach is a web-based mock interview platform that helps users practice interviews and improve their communication skills.

Features

* User Login & Signup
* Mock Interview Questions
* Voice Input Support
* Performance Scoring
* Modern Responsive UI
* Chrome & Firefox Compatible

Technologies Used

* HTML
* CSS
* JavaScript

How to Run

1. Download the project files.
2. Open login.html in your browser.
3. Create an account and start practicing interviews.

Future Enhancements

* AI-based Feedback
* Resume Analysis
* Marathi Interview Mode
* Job Recommendations
